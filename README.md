IMPORTANT SETUP STEPS (READ BEFORE PLAYING)
-------------------------------------------------------------------------------
1. WINDOWS DEFENDER WORKAROUND (False Positive)
Because this launcher handles custom networking and automated updates, Windows 
Defender may mistakenly flag it as a virus. We have contacted Microsoft to get 
the app officially whitelisted, but until they accept it, you must add an 
exclusion or Defender will delete the launcher:
  - Open Windows Settings -> Privacy & Security -> Windows Security.
  - Click 'Virus & threat protection', then click 'Manage settings'.
  - Scroll down to 'Exclusions' and click 'Add or remove exclusions'.
  - Click 'Add an exclusion' -> 'Folder', and select your Sparc game folder 
    (where this launcher is located).

2. WINDOWS FIREWALL
To connect to servers and use the custom Voice Chat, this app needs to send 
data through your local network:
  - The first time you run the app or join a game, Windows Firewall will pop up.
  - Check BOTH boxes (Private and Public networks) and click 'Allow Access'.
  - If you accidentally denied it, open the Windows Start Menu, search for 
    'Allow an app through Windows Firewall', click 'Change settings', 
    'Allow another app', and browse to the Sparc Reviver Client executable.

3. KEYBOARD LAYOUT (For Avatar Injection)
When you join a match, the launcher automatically types your Gamertag and 
custom Avatar data directly into the game's hidden developer console. 
  - For this to work correctly, your Windows Keyboard Layout must be set to a 
    standard English (US or UK) QWERTY layout. 
  - If you use a different language layout (like AZERTY or QWERTZ), Windows 
    might type the wrong special characters (like quotes or brackets), which 
    will cause your avatar to fail to load!
